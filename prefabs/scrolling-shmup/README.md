Pixel art to help you create a full 2D scrolling shooter themed around classic arcade games.

Based on the Scrolling SHMUP Asset Bundle
Genre: Retro Arcade Scrolling Shooter. 
Style: Pixel art, chiptune music and sound effects.
Animation: Yes
Objects: 3 small aircraft, 3 medium aircraft, 3 large aircraft, 1 player aircraft. 
Tilesets: Yes
SFX: Yes
Music: Yes
