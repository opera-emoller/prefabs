# Pixel Protagonists

Four heroic, pixel art protagonists for your next game featuring a Knight, a Monk, a Wizard and a Ninja.

Based on the Pixel Protagonists Asset Bundle
Genre: High defiition pixel character art
Style: Pixel art, shaded, highly detailed
Animation: Yes
VFX: Yes
Sprites: Idle, Run, Jump, Close Attack, Ranged Attack, Hit, Stun and Defeat animations for a Knight, Monk, Wizard and Ninja character. Projectile weapons, spell attacks and blood splatter VFX animations.
