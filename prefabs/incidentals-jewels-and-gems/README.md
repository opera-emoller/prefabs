Icons and images of Jewels and Gems that go perfectly with the GameMaker Match 3 game template project.

Specific Elements: Coins, diamonds, heart shapes, hexagons, keys, pentagons, bottle, rectangular, and triangle shapes in blue, cyan, green, orange, pink, purple, red and yellow.
